GetCurrentObject
VB声明
Declare Function GetCurrentObject Lib "gdi32" Alias "GetCurrentObject" (ByVal hdc As Long, ByVal uObjectType As Long) As Long
说明
用于获得指定类型的当前选定对象
返回值
Long，当前选定对象的句柄，零表示错误
参数表
参数 类型及说明
hdc Long，欲查询的设备场景
uObjectType Long，对象类型。可以是下述常数之一：OBJ_PEN，OBJ_BRUSH，OBJ_PAL，OBJ_FONT，或OBJ_BITMAP
