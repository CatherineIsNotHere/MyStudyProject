StrokePath
VB声明
Declare Function StrokePath Lib "gdi32" Alias "StrokePath" (ByVal hdc As Long) As Long
说明
用当前画笔描绘一个路径的轮廓。打开的图形不会被这个函数关闭
返回值
Long，TRUE（非零）表示成功，否则返回零。会将GetLastError设置为下述值：ERROR_CAN_NOT_COMPLETE，ERROR_INVALID_PARAMETER，ERROR_NOT_ENOUGH_MEMORY
参数表
参数 类型及说明
hdc Long，包含了要描绘和填充的那个路径的一个设备场景
