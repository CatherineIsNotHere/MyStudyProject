CreateBrushIndirect
VB声明
Declare Function CreateBrushIndirect Lib "gdi32" Alias "CreateBrushIndirect" (lpLogBrush As LOGBRUSH) As Long
说明
在一个LOGBRUSH数据结构的基础上创建一个刷子
返回值
Long，如执行成功，返回指向新刷子的一个句柄。零表示失败
参数表
参数 类型及说明
lpLogBrush LOGBRUSH
注解

如不再需要，请用DeleteObject函数删除刷子。也请参考CreateBrush函数，它的参数与LOGBRUSH结构的字段是对应的
