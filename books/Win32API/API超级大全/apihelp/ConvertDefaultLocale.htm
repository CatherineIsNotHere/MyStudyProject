ConvertDefaultLocale
VB声明
Declare Function ConvertDefaultLocale Lib "KERNEL32" Alias "ConvertDefaultLocale" (ByVal Locale As Long) As Long
说明
将一个特殊的地方标识符转换成真实的地方ID
返回值
Long，如执行成功，返回实际的地方ID。如失败则返回传递给Locale参数的值
参数表
参数 类型及说明
Locale Long，如设为常数LOCALE_SYSTEM_DEFAULT和LOCALE_USER_DEFAULT，可分别接收默认的系统或用户地方设置。如设为零，则取得语言方面的地方设置。主语言ID用于接收采用了默认子语言的地方信息
注解

用LOCALE_SYSTEM_DEFAULT和LOCALE_USER_DEFAULT这两个值调用函数时，得到的结果与GetSystemDefaultLCID 和 GetUserDefaultLCID函数是一样的
