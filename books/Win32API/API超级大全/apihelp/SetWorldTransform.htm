SetWorldTransform
VB声明
Declare Function SetWorldTransform Lib "gdi32" Alias "SetWorldTransform" (ByVal hdc As Long, lpXform As xform) As Long
说明
设置世界转换
返回值
Long，执行成功为非零值，失败为0
参数表
参数 类型及说明
hdc Long，设备场景
lpXform xform，一个包含将用世界转换的结构
注解

适用于Windows NT
在使用世界转换前必须调用SetGraphicsMode函数设置图形模式为GM_ADVANCED模式
