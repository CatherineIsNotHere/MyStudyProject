EnumDateFormats
VB声明
Declare Function EnumDateFormats Lib "KERNEL32" Alias "EnumDateFormats" (ByVal lpDateFmtEnumProc As Long, ByVal Locale As Long, ByVal dwFlags As Long) As Long
说明
列举指定的“当地”设置中可用的长、短日期格式
返回值
Long，TRUE（非零）表示成功，零表示失败。会将GetLastError设置设为下述某个常数：ERROR_BADDB，ERROR_INVALID_FLAGS，ERROR_INVALID_PARAMETER
参数表
参数 类型及说明
lpDateFmtEnumProc Long，指向为每种日期格式都调用的一个函数的指针。用Addressof运算符得到位于一个标准模块中的函数的地址
Locale Long，用于决定具体枚举格式的“地方”的设置
dwFlags Long，DATE_SHORTDATE 或 DATE_LONGDATE常数之一，分别用于枚举短或长日期格式
