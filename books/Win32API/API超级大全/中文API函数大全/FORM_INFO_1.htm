FORM_INFO_1――请参考AddForm函数
类型定义
Type FORM_INFO_1
Flags As Long
pName As Long
Size As SIZEL
ImageableArea As RECT
End Type
说明
Contains information about a print form.
字段表
字段 类型及说明
Flags Long，如表单内建到打印机，则为常数FORM_BUILTIN
pName Long，表单的名字
Size SIZEL，以千分之一毫米为单位的打印媒介尺寸
ImageableArea RECT，可以绘图的实际区域，以千分之一毫米为单位
