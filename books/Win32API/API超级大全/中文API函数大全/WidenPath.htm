WidenPath
VB声明
Declare Function WidenPath Lib "gdi32" Alias "WidenPath" (ByVal hdc As Long) As Long
说明
根据选定画笔的宽度，重新定义当前选定的路径。例如，假设路径描述了一个封闭的矩形。面积为10×10像素，而且用一个3像素宽的画笔描绘。此时，加宽后的路径将由一个12×12的矩形构成
返回值
Long，TRUE（非零）表示成功，否则返回零。会将GetLastError设置为下述值：ERROR_CAN_NOT_COMPLETE，ERROR_INVALID_PARAMETER，ERROR_NOT_ENOUGH_MEMORY
参数表
参数 类型及说明
hdc Long，包含路径的一个设备的句柄
注解

所有贝塞尔曲线都会由这个函数转换成线段
