PROVIDOR_INFO_1――请参考AddPrintProvidor函数
类型定义
Type PROVIDOR_INFO_1 ' 12 Bytes
pName As Long
pEnvironment As Long
pDLLName As Long
End Type
说明
This structure is used to describe a print providor
字段表
字段 类型及说明
pName String，供应商的名字
pEnvironment String，指定供应商的设计环境（比如“Windows NT x86”）。对于本地系统，设为vbNullString
pDLLName String，供应商使用的DLL的名字
