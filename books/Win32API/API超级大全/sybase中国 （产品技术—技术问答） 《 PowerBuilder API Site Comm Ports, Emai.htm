PowerBuilder API Site Comm Ports, Email and PowerBuilder
　　
　
　
　　There is a product produced by SaxSoft, Inc. which provides an external DLL, OCX/VBX that allows PowerBuilder 5.0 (32 bit) to communicate with the Comm Ports.

　　SaxSoft, Inc. at http://www.saxsoft.com provides the software, documentation and tech support to get their DLL to work with PowerBuilder. The cost is $495.00 and a sample can be downloaded for free. This customer noted that the OCX does not work correctly with PB and was able to get the DLL to be called as an external dll from PB 5.0 (32 bit).

　　Other companies to look into are: Cresent at 617-280-4203. Their products are OCX/VBX compliant and are about $175.

　　This one just in... IDS, Inc. This company has a mail enabling OCX.
