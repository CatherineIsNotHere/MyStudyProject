POINTAPI Type

Type POINTAPI
  x As Long
  y As Long
End Type

POINTAPI-type variables hold a point's coordinates. This type is used throughout the API for storing a point.

x
The x coordinate of the point.
y
The y coordinate of the point.

Used by: GetCursorPos, MoveToEx
