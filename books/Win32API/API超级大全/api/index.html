Windows 95 API Guide

Last Update: Friday, November 6, 1998

Number of Calls Listed: 74

Welcome to the Windows 95 API Guide! This is a list of many of the various API functions built into Windows 95. Of course, much of this applies to Windows NT and Windows 98. This will always be a work-in-progress, so don't expect a full and complete guide to the API. I do do my best, however.

If you are new to using the Windows API to add lots of functionality to your 32-bit Windows-based programs, or just want a nice little introduction, please read the Introduction Page. If you would like to save this entire guide to your computer for later reference, you can download a zipped copy of it. If you do, you should check back here every so often for updated and additions. Otherwise, please use one of the listings below to start your Windows API learning experience!

One more introductory note. I am only human, so there may be some mistakes in this guide. If it doesn't look right (or you know I'm just plain wrong), use it with caution. Better yet, e-mail me about it. I'd also appreciate it if you had any API information you would like for me to include in this guide.
